# Modulo en NPM

## Instrucciones para instalacion del modulo

```bash 
$npm install adventurer_races
``` 

## Uso del modulo:
``` js
    const Aventureros = require('adventurer_races');

    let frase = Aventureros.obtieneFrase(n); //Donde n es un numero del 0 al 4
```
